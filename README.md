



EchoVerse converts text and PDF files to emotional audiobooks with tone adaptation via AI.

 Features
- PDF to text extraction
- AI tone adaptation (Neutral, Suspenseful, Inspiring)
- Text-to-speech audio generation
- Downloadable MP3 files
- Pretty React interface

Quick Start
1. Clone repo: `git clone https://github.com/pallavika-sheripally/projhope.git`
2. Backend: `cd backend && pip install -r requirements.txt && python app.py`
3. Frontend:
`cd frontend && npm install && npm start`
4. Open: http://localhost:3000

Built With
- React.js Frontend
- Flask Backend
- IBM Granite AI Model
- Google Text-to-Speech
- Hugging Face API
